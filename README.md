# AzureGrupodeRecursos
Resumo sobre a aula disponibilizada pela DIO sobre grupos de recursos da azure.


Durante a aula foi informado como é possivel crirar um grupo de recursos para separar os recursos, como em setores diferentes. Algo como guardar suas coisas em caixas separadas.
Criado o grupo de recursos também é possivel cirar restrições e definir quem pode acessar esses recursos.
Além disso foi mostrada uma pequena previa de como criar uma Vnet
